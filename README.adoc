== Setup

=== JBoss EAP

==== configuration

. messaging subsystem changes
.. *disable security:*
<security-enabled>false</security-enabled>
.. *enable message counter:* 
<message-counter-enabled>true</message-counter-enabled>


==== Start-up

-----
cd $JBOSS_HOME
./bin/standalone.sh --server-config=standalone-full.xml
-----

==== Build and Deploy

----
cd hornetqDiagnostics
mvn clean install -DskipTests
cp mdb/target/mdb-1.0.jar $JBOSS_HOME/standalone/deployments
----
